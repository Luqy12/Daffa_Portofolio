Listrik Pasca Bayar - LSP

Sistem manajemen pembayaran listrik pasca bayar berbasis web. Agustus 2025. Teknologi: PHP, MySQL, HTML, CSS, JavaScript.

GitHub Repository Project Documentation

Project Overview

LSP (Listrik Pasca Bayar) adalah sistem manajemen pembayaran listrik pasca bayar yang dikembangkan untuk memudahkan pengelolaan data pelanggan, penggunaan listrik, tagihan, dan pembayaran. Sistem ini dirancang untuk membantu perusahaan listrik dalam mengelola operasional sehari-hari dengan efisien.

Fitur utama meliputi manajemen data pelanggan, pencatatan penggunaan listrik, pembuatan tagihan otomatis, sistem pembayaran, dan laporan lengkap. Sistem ini juga dilengkapi dengan fitur dark mode dan light mode untuk kenyamanan pengguna, serta kemampuan download laporan dalam format yang mudah dibaca.

Dibangun menggunakan PHP sebagai bahasa pemrograman backend, MySQL sebagai database, dan HTML/CSS/JavaScript untuk frontend. Proyek ini menunjukkan kemampuan dalam pengembangan aplikasi web full-stack dengan fokus pada fungsionalitas dan user experience.

Database Structure

Database LSP terdiri dari beberapa tabel utama: Admin, Level, Pelanggan, Penggunaan, Tagihan, Tarif, dan User. Setiap tabel dirancang untuk menampung data yang diperlukan dengan relasi yang tepat untuk menjaga integritas data.
